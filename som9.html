A resposta correcta é:
É o sotfware encargado de xestionar a instalación, eliminación,
actualización e mantemento dos paquetes/SW.
→ Xestor de paquetes,
É unha colección de arquivos binarios, de documentación ou de
configuración relacionados entre si por constituír unha aplicación ou
parte dunha aplicación. Existen distintos formatos de paquete.
→ Paquete,
É un lugar ( cdrom, servidor web, servidor FTP) onde se almacenan
paquetes, co propósito de facelos accesibles aos sistemas clientes e que
estes poidan, de modo sinxelo, descargalos e instalalos ou actualizalos.
→ Repositorio ou réplica,
Relación entre diferentes paquetes que se precisan entre si, e forman no
seu conxunto, os programas necesarios para o funcionamento dunha
aplicación.
→ Dependecias,
É un conxunto de aplicacións, organizadas en paquetes e accesibles a
través de servidores, que se estruturan de modo que se facilite a súa
instalación, mantemento e actualización.
→ Sistema de paquetes

A resposta correcta é:
Distribución que se actualiza aínda con maior celeridade e que serve como
alimentación á rama en probas.
→ unstable,
Sección onde se inclúen os paquetes que son software libre, pero ten
dependencia de paquetes que non son software libre.
→ contrib,
Sección onde se inclúen os paquetes que cumpren os requisitos para ser
considerados por Debian software libre e que, ademais, só teñen
dependencia de paquetes sowftware libre que tamén pertencen a este
compoñente.

Sección onde se inclúen os paquetes que que non son software libre a
criterio de Debian.
→ non-free,
Distribución orientada ao uso en servidores, durante cuxa vida só recibe
actualizacións para correción de bugs e problemas de seguridade.
→ stable,
Distribución que se actualiza regularmente con novo software e versións
máis recentes do software xa existente.
